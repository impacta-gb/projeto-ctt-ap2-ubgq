# Introdução e Instalação

Go, também conhecida como Golang, é uma linguagem de programação criada pelo Google com foco em simplicidade, desempenho e concorrência.

A linguagem é muito utilizada em:

- APIs

- Microsserviços

- Sistemas distribuídos

- Ferramentas de infraestrutura

- Cloud Computing

## Características da Linguagem

| Característica | Descrição |

|---|---|

| Simples | Sintaxe fácil de aprender |

| Rápida | Compilação eficiente |

| Concorrente | Suporte nativo a Goroutines |

| Segura | Tipagem forte |

## Instalação

A instalação pode ser feita através do site oficial:

https://go.dev/dl/

## Verificando instalação

go version

## Primeiro programa em Go

package main



import "fmt"



func main() {

&#x20;   fmt.Println("Olá Mundo")

}

Dica

O comando go run executa arquivos Go sem necessidade de compilação manual.
