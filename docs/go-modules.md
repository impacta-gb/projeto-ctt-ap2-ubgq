\# Go Modules



Go Modules são utilizados para gerenciamento de dependências.



\## Inicializando módulo



```go

go mod init projeto

```



\## Instalando dependência



```go

go get github.com/gin-gonic/gin

```



\## Arquivo go.mod



O arquivo `go.mod` armazena as dependências do projeto.



\## Atualizando dependências



```go

go mod tidy

```



!!! note "Importante"

&#x20;   O Go Modules facilita o controle de versões das bibliotecas.
