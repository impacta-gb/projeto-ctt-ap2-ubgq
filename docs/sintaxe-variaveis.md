\# Sintaxe Básica e Variáveis



A sintaxe da linguagem Go foi criada para ser simples e objetiva.



\## Declaração de Variáveis



```go

var nome string = "Giovanna"

```



\## Declaração com inferência



```go

idade := 23

```



\## Tipos básicos



| Tipo | Exemplo |

|---|---|

| int | 10 |

| float64 | 5.8 |

| string | "Olá" |

| bool | true |



\## Constantes



```go

const pi = 3.14

```



\## Função principal



```go

func main() {



}

```



!!! warning "Importante"

&#x20;   O operador `:=` só pode ser utilizado dentro de funções.
